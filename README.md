# Imperative Programming C Course

## Introduction

This repository contains code examples and exercises from an introductory course on imperative programming using the C programming language. C is a procedural (imperative) language known for its efficiency and fine-grained control over system resources (such as memory), making it ideal for learning how software works at a low level. The objective of this course is to teach fundamental programming concepts and problem-solving using C, focusing on writing step-by-step instructions that manipulate program state.

### Key topics include:

* Variables and data types – understanding how to store, retrieve, and manipulate data in programs.
* Control flow – using conditional statements and loops (e.g., if/else, for, while) to direct the execution path of the program.
* Functions – breaking the program into reusable pieces (defining and calling functions to perform specific tasks and improve code organization).
* Memory management – managing dynamic memory using pointers (e.g., allocating memory with malloc and freeing it with free) and understanding how to avoid memory leaks.
* Modularization – organizing code into multiple files or modules (using header files .h for shared declarations) to improve maintainability and clarity of larger programs.

## Structure

This repository is organized into the following sections/directories:

* **examples/** – Contains sample code examples that illustrate the concepts from the course.
* **exercises/** – Contains practice exercises for you to solve.
* **projects/** – Contains basic projects that integrate multiple concepts from the course.

## Prerequisites

* C compiler (GCC or equivalent)
* Terminal or command line
* (Optional) IDE like Code::Blocks, Dev-C++, or VS Code with C extension

## How to Use

```bash
# Clone the repository
git clone https://github.com/your-username/imperative-programming-c-course.git
cd imperative-programming-c-course

# Navigate to the example or exercise you want to run
cd examples/ # or exercises/ or projects/

# Compile and run (using GCC for example)
gcc file.c -o file
./file
```

You can also explore and edit the code to reinforce your understanding.

# Curso de Programação Imperativa em C

## Introdução

Este repositório contém exemplos de código e exercícios de um curso introdutório sobre programação imperativa utilizando a linguagem C. C é uma linguagem de programação procedural imperativa, conhecida por sua eficiência e pelo controle de baixo nível que oferece sobre os recursos do sistema (como a memória). Isso a torna ideal para aprender como o software funciona em um nível fundamental. O objetivo deste curso é ensinar conceitos básicos de programação e técnicas de resolução de problemas usando C, com foco em escrever instruções passo a passo que manipulam o estado do programa.

### Tópicos abordados incluem:

* Variáveis e tipos de dados – como armazenar, acessar e manipular dados em programas.
* Fluxo de controle – uso de estruturas condicionais e laços (if/else, for, while) para controlar a execução.
* Funções – organizar o programa em partes reutilizáveis para realizar tarefas específicas.
* Gerenciamento de memória – uso de ponteiros e alocação dinâmica com malloc/free.
* Modularização – separação do código em módulos com arquivos .h para declarações compartilhadas.

## Estrutura

* **examples/** – Exemplos de código com conceitos do curso.
